# &lt;CS/&gt; crowdshipped.dev

**You vote it. I build it.**

A community-driven micro-app platform where Instagram followers vote on app ideas. The most-liked idea (100+ likes) after each 21-day cycle gets built and deployed live.

🌐 [crowdshipped.dev](https://crowdshipped.dev) · 📸 [@crowdshipped.dev](https://instagram.com/crowdshipped.dev) · ✉️ [contact@crowdshipped.dev](mailto:contact@crowdshipped.dev)

---

## How It Works

1. **Comment** — Drop your app idea on the latest Instagram voting post
2. **Vote** — Community likes their favorite ideas for 21 days
3. **Win** — The most-liked comment wins (minimum 100 likes required)
4. **Ship** — The winning idea gets built and deployed to its own subdomain

No idea hits 100 likes? The attempt resets. Same app number, new round.
